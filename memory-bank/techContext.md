Technology Stack

Frontend
	•	Browser-based GUI

	•	SVG and Canvas-based rendering for EQ curves

Backend
	•	Local web server
	•	Audio DSP engine

GUI Specification

	•	Draggable EQ nodes
	•	Numeric input for precise control
	•	Visual feedback for gain, Q, and frequency
	•	Interactive multi-channel pipeline editor	

Style Guide (Web / CSS)
	•	Clean, minimal, professional aesthetic
	•	Dark-mode friendly color palette
	•	High contrast for readability
	•	Consistent spacing and typography

Technical Constraints
	•	LAN-only operation
	•	No external services or cloud dependencies
	•	Low-latency audio path
	•	Predictable resource usage

Error Handling and Logging
	•	Explicit logging for audio and UI subsystems
	•	Configurable error handling policy
	•	Clear separation between recoverable and fatal errors
